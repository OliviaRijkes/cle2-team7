## Installatie
1. Clone de repository
2. Maak lokaal een database aan (bijv. `bouw_reserveringen`)
3. Importeer `sql/schema.sql`
4. Pas `includes/db.php` aan indien nodig
5. Open `/db_test.php` om de databaseverbinding te testen
